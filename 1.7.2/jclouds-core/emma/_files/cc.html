50 * Creates type adapters for types handled in the following ways:
51 * <p/>
52 * <ul>
53 * <li>Deserialization</li>
54 * If there's an annotation designating a parameterized constructor, invoke that for fields correlating to named
55 * parameter annotations. Otherwise, use {@link ConstructorConstructor}, and set fields via reflection.
56 * <p/>
57 * Notes: primitive constructor params are set to the Java defaults (0 or false) if not present; and the empty object
58 * ({}) is treated as a null if the constructor for the object throws an NPE.
59 * <li>Serialization</li> Serialize based on reflective access to fields, delegating to ReflectiveTypeAdaptor.
60 * </ul>
61 * <h3>Example: Using javax inject to select a constructor and corresponding named parameters</h3>
62 * <p/>
63 * 
64 * <pre>
65 * 
66 * import NamingStrategies.*;
67 * 
68 * serializationStrategy = new AnnotationOrNameFieldNamingStrategy(
69 *    new ExtractSerializedName(), new ExtractNamed());
70 * 
71 * deserializationStrategy = new AnnotationConstructorNamingStrategy(
72 *    ImmutableSet.of(javax.inject.Inject.class),
73 *    ImmutableSet.of(new ExtractNamed()));
74 *    
75 * factory = new DeserializationConstructorAndReflectiveTypeAdapterFactory(new ConstructorConstructor(),
76 *      serializationStrategy, Excluder.DEFAULT, deserializationStrategy);
77 * 
78 * gson = new GsonBuilder(serializationStrategy).registerTypeAdapterFactory(factory).create();
79 * 
80 * </pre>
81 * <p/>
82 * The above would work fine on the following class, which has no gson-specific annotations:
83 * <p/>
84 * 
85 * <pre>
86 * private static class ImmutableAndVerifiedInCtor {
